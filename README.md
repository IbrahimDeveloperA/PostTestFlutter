## 📌 Инструкция работы приложения

### 🏠 Главный экран – **список постов (PostsListScreen)**
- При старте загружается список постов.  
- Есть **Pull-to-Refresh** (обновление свайпом вниз).  
- Поле поиска позволяет **фильтровать посты по заголовку**.  
- Посты отображаются в карточках (**PostCard**).  
- При клике на пост открывается экран **деталей**.  

### 📄 Экран деталей поста – **DetailPostScreen**
- Показывает **заголовок** и **текст** выбранного поста.  
- Использует **BLoC** для загрузки данных по `id`.  

### ⚙️ Экран настроек – **SettingScreen**
- **Переключатель темы** (светлая/тёмная).  
- После изменения темы обновляется всё приложение.
